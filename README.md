# Brain-Tumor-Detector
▪ Developed a deep learning-based image classification system to detect brain tumors from MRI scans using a Convolutional Neural Network (CNN).

▪ Implemented image preprocessing techniques including resizing, normalization, and channel transformation to prepare data for model training.

▪ Built a custom dataset pipeline with training and validation splits to ensure proper performance evaluation.

▪ Trained the model using the Adam optimizer and binary classification loss function, achieving an optimized validation with high accuracy.

▪ Evaluated model performance using accuracy metrics and loss analysis to monitor training behavior and detect overfitting.
